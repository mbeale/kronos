# kronos

A LISP with first class time series. Inspired by http://www.buildyourownlisp.com.

## Install

Nope

## Build

I've only ever built on Mac OSX.  

Windows? Nope

Build

```
cc -std=c99 -Wall kronos.c mpc.c -ledit -lm -o kronos
```

Run Repl

```
./kronos
```

Or run a script

```
./kronos mylisp.lisp
```
